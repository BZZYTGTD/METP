METP （Multi-function Execution Test Platform）是一个多功能自动化测试执行可视平台
可满足Monkey， Uiautomator， Robotium等多种（如有新需求可以扩展）自动化测试执行，结果处理，等需求
其优点在于
1. 可视化界面，更易操作，也减少了配置文件传参方式的繁琐和易出错
2. 自动识别插在PC上的手机，自动反射出测试用例，可以选择，排序执行
